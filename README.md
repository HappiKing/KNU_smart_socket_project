## 파일 설명

### `data.py`
- **목표**: 랜덤 값을 생성하여 DB에 저장한다.(추후 실제 전류 값으로 대체)
- **기능**: data를 제어하는 백엔드 역할을 한다.

### `page.py`
- **목표**: UI/UX를 담당한다.
- **기능**: 아이콘 클릭시 멀티탭을 켜고 끄는 기능을 제어한다.
